Develop a real-time face mask detection system using transfer learning.

## Dataset
Face Mask Dataset

Classes:
- WithMask
- WithoutMask

## Model
- MobileNetV2
- Transfer Learning
- Adam Optimizer

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

Accuracy: 98%+
